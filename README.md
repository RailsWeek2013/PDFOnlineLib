pdf_online_lib
==============

A small online library for pdf-texts.

Features:

Core:
PDFs hochladen, downloaden, (per Viewer ansehen)

0. Userverwaltung
	- Favoriten verwalten
	- User Information

0. PDFs privat/public flaggen
1. PDFs bewerten
1. Tag Search

2. Rezessionen schreiben
2. Public PDFs Gallery



Optional:
Wörtersuche in PDFs


Gems:
Devise
AuthLogic
paperclip
carrierwave
